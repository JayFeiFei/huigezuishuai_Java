# huigezuishuai_Java

1. #### 封装圆形类(Circle)

  1. 属性: 半径
  2. 方法: 计算圆形面积(当半径为负值时, 抛出自定义异常)

  ```java
  Main.java
  public class Main {
      public static void main(String[] args) {
          // write your code here
          Circle c = new Circle(-3);
          try {
              System.out.println(c.square());
          } catch (CircleException e) {
              System.out.println(e.getMessage());
          }
      }
  
  }
  
  CircleException.java
  public class CircleException extends Exception {
      public CircleException(String message) {
          super(message);
      }
  }
  
  
  Circle.java
  public class Circle {
  
      private double r;
  
      public Circle() {
      }
  
      public Circle(double r) {
          this.r = r;
      }
  
      public double getR() {
          return r;
      }
  
      public void setR(double r) {
          this.r = r;
      }
  
      @Override
      public String toString() {
          return "Circle{" +
                  "r=" + r +
                  '}';
      }
  
      double square() throws CircleException {
          if (r < 0) {
              throw new CircleException("输入的半径不能为负数");
          }
          return Math.PI * r * r;
      }
  }
  ```

2. #### 封装银行类(Bank)

   1.  属性: 金额
   2. 方法: 取款(如果取款的数额大于存款, 抛出自定义异常; 当取款数为负数时, 抛出自定义异常)

   ```java
   Main.java
   public class Main {
       public static void main(String[] args) {
           Bank bank = new Bank(100);
   
           try {
               bank.withdrawal(150);
           } catch (InsufficientFundsException e) {
               e.printStackTrace();
           } catch (NagativeFundsException e) {
               e.printStackTrace();
           }
   
           try {
               bank.withdrawal(-10);
           } catch (InsufficientFundsException e) {
               e.printStackTrace();
           } catch (NagativeFundsException e) {
               e.printStackTrace();
           }
       }
   }
   
   InsufficientFundsException.java
   public class InsufficientFundsException extends Exception {
       @Override
       public String getMessage() {
           return "余额不足";
       }
   }
   
   NagativeFundsException.java
   public class NagativeFundsException extends Exception {
       @Override
       public String getMessage() {
           return "取款不能为负数";
       }
   }
   
   Bank.java
   public class Bank {
       private double balance;
   
       public Bank() {
       }
   
       public Bank(double balance) {
           this.balance = balance;
       }
   
       public double getBalance() {
           return balance;
       }
   
       public void setBalance(double balance) {
           this.balance = balance;
       }
   
       @Override
       public String toString() {
           return "Bank{" +
                   "balance=" + balance +
                   '}';
       }
   
       public void withdrawal(double amount) throws InsufficientFundsException, NagativeFundsException {
           if (amount > balance) {
               throw new InsufficientFundsException();
           }
           if (amount < 0) {
               throw new NagativeFundsException();
           }
           balance -= amount;
       }
   }
   ```
